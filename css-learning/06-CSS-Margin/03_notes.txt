CSS MARGIN - QUICK NOTES
=========================

WHAT IS MARGIN?
---------------
Space OUTSIDE an element's border
Creates distance between elements
Transparent (invisible)


MARGIN PROPERTIES
------------------

1. All Sides Same:
   margin: 20px;

2. Individual Sides:
   margin-top: 10px;
   margin-right: 20px;
   margin-bottom: 30px;
   margin-left: 40px;

3. Shorthand - Two Values:
   margin: 20px 40px;
   (Top/Bottom Left/Right)

4. Shorthand - Four Values:
   margin: 10px 20px 30px 40px;
   (Top Right Bottom Left)
   
   Remember: TRouBLe
   T = Top
   R = Right
   B = Bottom
   L = Left


CENTER ELEMENT
--------------
margin: 0 auto;

Requirements:
✓ Element must have WIDTH set
✓ Element must be BLOCK-level

Example:
.box {
    width: 500px;
    margin: 0 auto;  /* Centers horizontally */
}


NEGATIVE MARGIN
---------------
margin-top: -20px;  /* Pulls element UP */
margin-left: -10px; /* Pulls element LEFT */

Use carefully! Can overlap elements.


MARGIN COLLAPSE
---------------
Vertical margins between elements COLLAPSE:
- Larger margin wins
- Only happens with vertical margins (top/bottom)
- Does NOT happen with horizontal margins

Example:
Box1: margin-bottom: 30px;
Box2: margin-top: 20px;
Result: 30px gap (not 50px!)

COMMON PATTERNS
---------------

1. Space Around Element:
   margin: 20px;

2. Center Element:
   margin: 0 auto;
   width: 500px;

3. Space Between Sections:
   margin: 40px 0;

4. Remove Default Margin:
   margin: 0;

5. Card Spacing:
   margin: 20px;


WHEN TO USE MARGIN
------------------
✓ Space between elements
✓ Center elements horizontally
✓ Create section spacing
✓ Push elements apart
✓ Layout adjustments


MARGIN VS PADDING
-----------------
MARGIN:
- Outside border
- Creates space between elements
- Transparent
- Can be negative

PADDING:
- Inside border
- Creates space inside element
- Uses element's background color
- Cannot be negative


IMPORTANT TIPS
--------------
✓ Use margin for space BETWEEN elements
✓ Use padding for space INSIDE elements
✓ margin: 0 auto centers horizontally
✓ Vertical margins collapse
✓ Always set width when centering
✓ Use margin for layout spacing
